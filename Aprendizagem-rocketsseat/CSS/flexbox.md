# Flexbox
* Nos permite possiocnar os elementos detnro da caixa.
* Controle em uma dimensão (Horizontal ou vertical) 
* Alinhamento, direcionamento, ordenar e tamanhos.

```css
div.parent {
    display: flex;
}
``` 

## flex-direction

* Qual a direção do flex: horizontal(x) ou vertical(y)
* row | column

## Alinhamento

* justify-content
* align-items
